# Blog Project

Este é um projeto de blog construído com AdonisJS, TypeScript, Tailwind CSS, SSR (Server-Side Rendering) e Alpine.js.

## Funcionalidades

- Criação de usuário
- Login
- Autenticação
- Permissões de usuário
- Criação de post
- Comentários em posts
- Respostas a comentários
- Telas responsivas
